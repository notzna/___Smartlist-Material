# SmartList sign-in mock

This folder contains a static implementation of the **SmartList** sign-in screen based on the provided Figma design.

## Running locally

1. Open this folder in a browser:
   - Option 1: Double–click `index.html` in your file explorer.
   - Option 2: Start a simple static server (for example, with `npx serve .`) and open the served URL.

2. The sign-in card is centered on the page and matches the Figma layout and typography as closely as possible using plain HTML and CSS.

## Tech stack

- Plain `index.html` + `styles.css`
- No build tooling or dependencies required

If you want this screen recreated in a specific framework (React, Next.js, Vue, etc.), let the AI know and it can convert this markup and styling to match your stack.
